Simulated Dispatch Queue

The duty, delivered.

BNSS s.479(3) requires the Superintendent of jail to apply to the Court "forthwith" when a prisoner crosses the custody threshold — a duty that can only be discharged if someone is told, case by case, the moment it crystallises. These are the messages a production system would dispatch, generated live from the same rules engine as the dashboard.

Why an escalation ladder, not a single reminder

Each stage maps to a different accountable role: the Superintendent holds the s.479(3) duty; the DLSA holds the legal-aid mandate; the UTRC chairperson holds review oversight. A message that isn't acted on doesn't disappear — it climbs. The audit trail this creates ("duty crystallised on date X; application filed / not filed by date Y") is itself the accountability mechanism.
